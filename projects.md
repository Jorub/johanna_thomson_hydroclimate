# Projects & Code

Open research software and reproducible materials are central outputs of my work. The projects below support transparent analysis of evapotranspiration, precipitation, and terrestrial water-cycle datasets.

## evapoRe

An R-based application and package for exploratory analysis of evapotranspiration datasets. The software supports structured comparison and visualization of multiple products.

[CRAN package](https://cran.r-project.org/package=evapoRe) · [Associated article](https://doi.org/10.1016/j.envsoft.2026.106884)

## Evapotranspiration topology framework

Reproducible materials for comparing recent terrestrial evapotranspiration trends across multiple data sources using a topology framework.

[GitHub repository](https://github.com/Jorub/egu2026-et-topology-framework)

## ITHACA

Collaborative research on uncertainty in terrestrial water-cycle estimates, including comparisons of global precipitation datasets and analysis of agreement across environmental settings.

[Project code](https://github.com/imarkonis/ithaca) · [Project overview](https://yannismarkonis.wordpress.com/ithaca/)

### Precipitation dataset selection

Code supporting the assessment of precipitation products for hydrological studies. Data access is available through the `download_data()` function in the pRecipe R package.

[Analysis code](https://github.com/imarkonis/ithaca/tree/main/projects/uncertainty_prec) · [pRecipe on CRAN](https://cran.r-project.org/package=pRecipe)

### Spatial partitioning of precipitation agreement

Code and archived data for analysing how agreement among terrestrial precipitation datasets varies across environmental conditions.

[Analysis code](https://github.com/imarkonis/ithaca/tree/main/projects/partition_evap) · [Archived data](https://zenodo.org/records/10836849)

## Potential evapotranspiration method sensitivity

Data and code for assessing how potential evapotranspiration method selection affects trends in hydrological cycle components across Europe.

[Zenodo archive](https://doi.org/10.5281/zenodo.14008649) · [Associated article](https://doi.org/10.5194/hess-29-4395-2025)

## Working principles

- Version-controlled analysis and transparent provenance
- Reusable R workflows and documented data access
- Archived code and data linked to publications
- Clear treatment of methodological choices and uncertainty

More repositories are available on [GitHub](https://github.com/Jorub).
